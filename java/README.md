### Java의 특징
>운영체제(하드웨어와 소프트웨어의 중간역할을 하는 프로그램)
>1. 운영체제에 구애받지 않는다. 한번만 작성하면 어디서든 실행된다.
>2. 객체지향 프로그래밍(OOP)<br>
Object-Oriented-Programming
>3. 자동 메모리 관리 <br>
>메모리 -> 컴퓨터 RAM 부품
 프로그램을 실행하면 메모리의 일정량을 차지한다.
 프로그램 내부에서 사용되지 않는 것들을 자바가 알아서 제거

### 변수란?
 >변수 : 프로그램에서 데이터를 저장하기 위해 메모리 공간에 이름을 부여한 것
 
### 기본자료형 vs 참조자료형

 >자료형 : 데이터의 재질과 크기<br>
 변수에 저장할 데이터의 종류와 크기를 정의하는 것<br>
ㄴ 기본 자료형 : 값 자체를 들 고 있다<br>
ㄴ 참조 자료형 : 값이 있는 주소값을 알고 있다
 
### 메서드란?
