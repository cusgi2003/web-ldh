### 변수란?

### 기본자료형 vs 참조자료형

>자료형 : 데이터의 재질과 크기<br>
변수에 저장할 데이터의 종류와 크기를 정의하는 것<br><br>
기본 자료형 : 값 자체를 들 고 있다<br>
참조 자료형 : 값이 있는 주소값을 알고 있다
 

### 메서드란?
>메서드 : 클래스 안에 정의된 함수

위에서 아래로, 좌에서 우로 동작한다 (대입 연산자는 오른쪽에서 왼쪽으로 동장)


### 조건문
>조건식 : 연산의 결과가 true 또는 false로 반환되는 식<br>
(비교연산자, 논리연산자)
>1.  if문<br>
if -> 조건에 따라 실행할지 말지 결정<br>
if - else -> 참일 때 실행할 명령, 거짓일 때 실행할 명령<br>
if - else - if -> 여러 조건을 비교하여, 실행할 명령을 결정<br><br>
>2. switch문<br>
비교값과 일치하는 조건값의 case로 접근하여 바로 실행<br>
Java 12부터는 switch를 연산식(expression)으로 사용할 수 있다.


### 반복문
>조건에 따라 명령을 반복할지 말지 결정하는 문법<br>
> 1. for문<br>
반복횟수가 정확하게 정해져 있을때<br><br>
> 2. 이중 for문<br>
for문 안에 for문이 있는 형태 <br>
바깥쪽 for문이 한바퀴 돌 때 안쪽 for문은 반복을 전부 수행한다.<br><br>
> 3. while문<br>
반복횟수를 정확히 모를때도 사용할 수 있다.<br><br>
> 4. do-while문<br>
while문과 같이 조건을 만족할 때까지 반복한다.<br>
while문과 다른 점은 먼저 루프를 한 번 실행한 후 조건식을 검사한다는 점이다.


### 기타 제어문
>반복문은 조건이 거짓이 될 때까지 반복을한다.<br>
내가 원하는 조건에 도달하면 반복횟수가 남았더라고 반복문을 끝내거나,<br>
다음 횟수로 건너뛰게 만들 수 있다.<br>
> 1. break문<br>
반복문 안에서 break를 만나게 되면 그 즉시 반복문을 종료한다<br><br>
> 2. continue문<br>
반복문 안에서 continue를 만나게 되면 증감식으로 간다.


### 메모리의 영역
> 1. 메소드 영역<br>
바이트코드 파일을 읽은 내용을 저장되는 영역<br>
클래스별로 상수, 정적필드,메소드코드, 생성자 코드등이 저장된다.
> 2. 힙 영역<br>
객체가 생성되는 영역이다.
> 3. 스택 영역<br>
변수들이 만들어지면 저장되는 영역이다.
> 4. null<br>
참조 타입 변수는 아직 주소를 저장하고 있지 않다는 뜻이다.


### 배열
>자료형을 데이터를 여러개 저장할 수 있는 저장공간<br>
배열을 사용하면 같은 자료형의 데이터를 효율적으로 다룰 수 있다.<br><br>
배열 선언 시 크기를 지정한다<br>
배열 선언 후 공간의 크기를 늘리거나 삭제할 수 없다.<br>
같은 자료형의 값만 저장할 수 있다.
