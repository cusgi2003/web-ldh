### Java의 특징
>운영체제(하드웨어와 소프트웨어의 중간역할을 하는 프로그램)
>1. 운영체제에 구애받지 않는다. 한번만 작성하면 어디서든 실행된다.
>2. 객체지향 프로그래밍(OOP)<br>
Object-Oriented-Programming
>3. 자동 메모리 관리 <br>
>메모리 -> 컴퓨터 RAM 부품
 프로그램을 실행하면 메모리의 일정량을 차지한다.
 프로그램 내부에서 사용되지 않는 것들을 자바가 알아서 제거


### 변수란?
 >변수 : 프로그램에서 데이터를 저장하기 위해 메모리 공간에 이름을 부여한 것
 

### 기본자료형 vs 참조자료형

>자료형 : 데이터의 재질과 크기<br>
변수에 저장할 데이터의 종류와 크기를 정의하는 것<br><br>
기본 자료형 : 값 자체를 들 고 있다<br>
참조 자료형 : 값이 있는 주소값을 알고 있다
 

### 메서드란?
>메서드 : 클래스 안에 정의된 함수

### 조건문
>조건식 : 연산의 결과가 true 또는 false로 반환되는 식<br>
(비교연산자, 논리연산자)
>1.  if문<br>
if -> 조건에 따라 실행할지 말지 결정<br>
if - else -> 참일 때 실행할 명령, 거짓일 때 실행할 명령<br>
if - else - if -> 여러 조건을 비교하여, 실행할 명령을 결정<br>
>2. switch문<br>
비교값과 일치하는 조건값의 case로 접근하여 바로 실행<br>
Java 12부터는 switch를 연산식(expression)으로 사용할 수 있다.


### 반복문
>조건에 따라 명령을 반복할지 말지 결정하는 문법<br><br>
for문   : 반복횟수가 정확하게 정해져 있을때<br>
while문 : 반복횟수를 정확히 모를때도 사용할 수 있다.
