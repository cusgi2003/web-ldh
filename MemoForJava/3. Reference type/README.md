## 메모리 영역

### 메서드 영역
```
  바이트 파일을 읽느 내용이 저장되는 영역
  클래스별로 상수, 정적 필드, 메서드 코드, 생성자 코드 등이 저장
```
### 힙 영역
```
  객체가 생성되는 영역
```
### 스택 영역
```
  변수들이 만들어지는 영역
```
## null

  아무것도 참조하지 않는 상태

## 메서드

  객체가 가지고 있는 기능을 실행하는 코드
  객체를 만든 두 점을 찍고 메서드 이름을 작성

  ```
    equals()     -> 문자열 값 비교
    charAt()     -> 문자 추출
    length()     -> 문자열 길이
    replace()    -> 문자열 대체
    substring()  -> 문자열 잘라내기
    indexOf()    -> 위치 찾기
    split()      -> 문자열 분리
  ```

## 배열

  같은 자료형의 데이터를 여러개 저장할 수 있는 저장공간

### 배열의 선언
  ```
    자료형[] 배열명;
    자료형 배열명[];

    타입[] 변수 = new 타입 [길이];
  ```
### 배열의 구조
  
  인덱스
  ```
    배열은 각 공간마다 위치를 알려주는 위치 값
    배열의 공간마다 붙여진 번호로 0부터 시작하여 순차적으로 증가
  ```

### 다차원 배열

  배열의 요소로 또 다른 배열을 가지는 것
  ```
    int [] [] arr = {{1,2}, {3,4}, {5,6}};
  ```

### 향상된 for문

  카운터 변수와 증감식을 사용하지 않고, 항목의 개수만큼 반복한 후 자동으로 for문을 빠져나감
  ```
    for(자료형 변수 : 배열){
      실행코드
    }
  ```
