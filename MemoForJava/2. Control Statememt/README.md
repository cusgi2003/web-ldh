## 조건문

  조건식에 따라서 프로그램 흐르을 제어할 수 있는 문법
  조건식의 true / false에 따라 결과가 결정됨

  ### if - else문

        if(조건식){
          조건식이 참일 때 실행할 문장.
        } else {
          조건식이 거짓일 때 실행할 문장.
        }

  ### if - else if

  비교해야할 조건이 여러개 있는 경우 사용하는 문법
  가독성을 높이기 위한 방법

        if(조건식1) {
          조건식1이 참일 때 실행할 문장
        } else if (조건식2) {
          조건식 1이 거짓이고 조건식 2가 참일 때 실행할 문장
        } else {
          위의 조건이 모두 거짓일 때 실행할 문장
        }
  else if문의 개수에는 제한이 없음
  마지막에 작성하는 else 블록은 필요없다면 생략이 가능

  ### if문의 중첩

  제어문은 자유롭게 중첩사용이 가능

        if(조건식1) {
          if(조건식2) {
            조건식 1, 2가 모두 참일 때 실행할 문장
          }
        }

  ### switch문

  if문과 비슷하지만 if문은 괄호안에 인자값이 true, false로 결정되는 조건식
  switch문에 조건으로 사용할 수 있는 연산자는 '==' 뿐
  두 개의 피연산자 값이 같을 때만 조건으로 활용

        switch(비교값) {
        case 조건값1 :
            비교값과 조건값이 일치할 때 실행할 문장
            break;
        case 조건값2 : 
            비교값과 조건값이 일치할 때 실행할 문장
            break;
        default : 
            비교값과 일치하는 조건값이 없을 때 실행;
        }


## 반복문

### for문

  주로 반복 횟수가 정해져 있을 때 사용하는 문법

        for(초기식; 조건식; 증감식) {
          조건식이 참일때 반복할 명령
        }
  
### while문

  반복 횟수가 정해져 있지 않고, 조건식이 treu일 경우 계속 반복하는 문법

        while(조건식){
          조건식이 참일때 반복할 명령
        }

### do-while문

  먼저 루프를 한 번 실행한 후 조건식을 검사
  조건식 결과와 상관없이 한 번은 실행

        do{
	        조건식이 참일 때 실행할 명령;
        }while(조건식);
