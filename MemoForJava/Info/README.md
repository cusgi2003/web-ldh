## 객체 

  현실 세계의 사물의 상태(속성) 행동(기능) 데이터화 시킨 것
  즉, 현실 세계의 대상을  프로그래밍적으로 모델링한 결과물


## 메모리

  데이터를 저장하고, 프로그램을 실행하는 공간
  프로그램이  실행되는 동안만 데이터를 저장


## 변수

  하나의 값을 저장할 수 있는 메모리 번지에 붙여진 이름

  변수의 선언
  ```java
    자료형 변수명
    int age;
  ```
  초기값은 변수선언과 동시에 대입할 수 있음
  ```java
    int age = 25;
  ```
  ### 변수명 명명 규칙
  ```
    1. 숫자가 먼저 들어가면 안된다
    2. 영어 대소문자를 구분한다
    3. 일반적으로 영어소문자로 시작한다
    4. _, $를 제외하고 특수기호가 포함될 수 없다
    5. 예약어 금지 (switch, while 등)
    6. 한글은 사용하지 말 것
  ```

## 형변환

  ### 자동형변환

    서로 다른 자료형간의 대입이나 연산을 할 때 형 변환으로 자료형을 일치시켜야 하지만 다음과 같은 경우는
    자동으로 형 변환을  해주기 때문에 생략 가능

    byte -> short -> int -> long
    float(4바이트) -> double(8바이트)

  ### 강제형변환

    큰 자료형에서 작은 자료형을 옮길 때 데이터의 손실이 발생할 수도 있고 아닐 수도 있음
    데이터의 손실이 발생하면 정확한 연산을 못하므로 강제형변환은 항상 주의

    (원하는 자료형) 데이터or 변수;
