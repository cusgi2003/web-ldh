윈도우 명령어

파일/폴더 관리 명령어
    ㄴ mkdir 폴더명 : 폴더의 새엉
    ㄴ cd 경로 : 폴더의 이동
    ㄴ dir : 현재 폴더의 파일 목록 출력
    ㄴ del 파일명 : 선택한 파일 삭제

  ㄴ rmdir 폴더명 /s /q
      ㄴ /s : 내부 파일까지 삭제
      ㄴ /q : 확인 없이 삭제
  ㄴ ipconfig : ip주소 확인
  ㄴ tesklist : 실행중인 프로세스확인
  ㄴ whoami : 현재 사용자 확인

  ㄴ echo
      ㄴ echo 내용 : 입력한 내용을 그대로 출력하는 명령어
      ㄴ echo. : 빈 줄 출력
      ㄴ > : 덮어쓰기
      ㄴ >> : 추가쓰기

  ㄴ type
      ㄴ 파일에 들어있는 내용을 터미널에 출력하는 기능
      ㄴ 설정파일 확인, 간단한 텍스트 내용 확인
      
      ㄴ type 파일명1, 파일명 2 : 여러개의 파일을 이어서 출력할 수 있다
      ㄴ type 파일명1 > 파일명2 : 파일명1의 내용을 파일명2로 저장할 수 있다
      ㄴ tyoe 파일명1, 파일명2 > 파일명3 : 두 파일을 하나로 합친다
      ㄴ type con : 키보드에서 값을 입력받아 출력, ctrl + z 누르고 엔터치면 종료

리눅스 명령
  ㄴ ls : 현재 위치의 파일 및 디렉터리 목록 출력
      ㄴ ls -l : 상세 정보 출력
      ㄴ ls -a : 숨김파일까지 출력

  ㄴ cd : 디렉터리 이동
  ㄴ rmdir : 폴더 삭제 ( 내용이 있는 폴더 삭제 -> rm -r 폴더명 )
  ㄴ cat : 파일 내용을 출력 ( cat 파일명 )

  ㄴ touch : 파일을 새로 만들거나, 기존 파일의 시간 정보를 변경할 때 사용하는 명령어
      ㄴ touch -c 파일명 : 파일이 없으면 생성하지 마라
      ㄴ touch -t 202604211130 파일명 : 내가 원하는 시간을 지정할 수 있다
      ㄴ touch -d "2026-04-21 11:30:00" 파일명 : 사람이 읽기 쉬운 날짜 형식 사용

  사용자 추가하기
      ㄴ useradd / adduser
      ㄴ 우분투 계열에서는 adduser

  사용자 비밀번호 설정
      ㄴ sudo passwd 아이디
  
  사용자 정보 확인하기
      ㄴ id 아이디

  사용자 삭제하기
      ㄴ sudo userdel 아이디

  홈 디렉터리까지 삭제
      ㄴ sudo userdel -r 아이디

  그룹 추가하기
      ㄴ sudo groupadd 그룹명

  그룹 삭제하기
      ㄴ sudo groupdel 그룹명

  사용자를 그룹에 추가하기
      ㄴ sudo usermod -aG 그룹명 아이디
        ㄴ -a : append, 기존그룹은 유지하고 추가
        ㄴ -G : 보조 그룹 지정

  사용자를 그룹에서 제거하기
      ㄴ sudo gpasswd -d 아이디 그룹명

  사용자에게 실행권한 추가 
      ㄴ chmod u+x test.txt

 그룹의 쓰기 권한 제거
      ㄴ chown g-w memo.txt

 기타 사용자 읽기 권한 제거
      ㄴ chmod o-t memo.txt

 sudo chown 새소유자 파일명

 디렉터리에서 하위 파일까지 한 번에 소유권 변경하기
     ㄴ sudo chown -R 사용자:그룹명 디렉토리명
