## 개발 기술 환경 정의
- 개발기술환경을 정의할 때 운영체제, DBMS, 미들웨어, 오픈소스 순으로 살펴본다.

| 구분 | 설명 |
| --- | --- |
| 운영체제 | 시스템이 실행되는 기본 소프트웨어 환경 |
| DBMS | 데이터를 저장, 조회, 수정, 삭제하는 데이터베이스 관리 시스템 |
| 미들웨어 | 운영체제와 애플리케이션 사이에서 기능을 지원하는 소프트웨어 |
| 오픈 소스 | 공개된 소스 코드를 활용하는 소프트웨어 |

### 운영체제의 정의
```
  컴퓨터의 하드웨어와 소프트웨어 자원을 관리하고, 프로그램이 실행될 수 있도록
  공통 서비스를 제공하는 소프트웨어
  컴퓨터와 프로그램 사이에서 중간 관리자 역할을 하는 기본 소프트웨어
```
### 운영체제 선정시 고려사항
```
  신뢰도
  성능
  기술 지원
  주변 기기
  구축 비용
```
