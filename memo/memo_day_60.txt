Reset CSS
브라우저가 자동으로 넣어주는 기본 여백, 목록, 점, 링크 밑줄, 버튼 모양 등을 제거해서
모든 브라우저에서 비슷한 출발점으로 디자인 하기 위한 CSS

Normalize CSS
브라우저의 기본 스타일을 완전히 제거하는 것이 아니라, 브라우저마다 다르게 적용되는 부분을 통일해주는 CSS

사용 방법
1. 태그에 직접 지정하는 방법
2. CDN

중앙정렬
- 화면이나 부모 요소 기준으로 콘텐츠를 가운데 배치하는 기술
- 웹페이지의 레이아웃에서 가장 많이 사요하는 정렬방식 중 하나이다.
- 버튼, 팝업, 로그인 폼 등 주요 요소를 강조할 떄 많이 사용한다.

flexbox : 자식 요소를 내가 원하는데로 정렬할 수 있는 속성
flex를 설정하는 주체를 부모요소 또는 컨테이너
속성에 영향을 받는 요소를 item이라고 한다.

컨테이너 관련 속성
flex-direction : 주축을 결정
flex-wrap : 가로 너비가 좁아지면 밑으로 내릴거냐
justify-content : 주 축에서 요소들의 정렬 방법을 설정한다.
align-items : 교차 축에서 요소들의 정렬 방법을 선택
