배열
- 여러개의 데이터를 한 번에 보관할 수 있는 자료구조

배열의 생성 방법
1. [] : 리터럴을 사용하는 방법
2. 생성자를 이용하는 방법
ㄴ 길이를 지정할 수 있다.
ㄴ 매개변수자리에 여러개의 값을 넣거나, 문자열을 넣으면 요소로 포함하는 배열을 생성

js배열의 특징
- 다양한 타입을 하나의 배열에 요소로 넣을 수 있다.
- 길이의 제한이 따로 없어서 넣는만큼 들어간다

- 리터럴로 만ㄷ르면 length가 0이다.
- 생성자를 통해서 초기 길이를 지정하면 공간만큼 empty가 들어있다

객체(object)
- 관련된 데이터(필드)와 동작(메서드)을 하나로 묶은 데이터 구조다.

객체의 생성
js에서는 클래스 없이 객체를 만드는 것이 가능하다.

1. 객체 리터럴 생성 방식
let 객체명 = {};

- 프로퍼티(property) : 객체 안에 들어있는 데이터
ㄴ 프로퍼티의 구조 -> 키 : 값

let user = {
	name : "홍길동",
	age : 30,
	address : "인천시",
	introduce : function() { 함수 안에서 실행하고자 하는 명령어; }
	introduce(){ 함수 안에서 실행하고자 하는 명령; }
}

객체 안에 있는 속성 값을 가져오거나 메서들르 실행하는 방법
객체명.속성명
객체명["속성명"]

객체명.메서드명();

생성자 함수를 이용하여 객체 만들기
js는 메서드 말고 독립적으로 호출하는 함수가 존재하기 때문에 가능
function Student() {
	this.name = name;
	this.age = age;
	this.address = address;
}

let s1 = new Student('홍길동', 20, '인천시');

클래스를 통한 객체 생성
- 여러개의 객체를 만들 때 편하다

class Member{
	constructor(name, age) {
		this.name = name;
		this.age = age;
	}
}

let m1 = new Member('홍길동',  20);
let m2 = new Member('김길동', 20);
