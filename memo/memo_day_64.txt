속성 선택자
특정 속성을 가진 요소를 선택
input[type] : input태그 중에서 type 속성을 가진 태그를 선택
input[type ="text"] : input 태그 중에서 type ="text"를 가진 태그를 선택

BEM 방법론
클래스 이름을 너무 일반적으로 작성하면 겹칠 수 있으니까 역할을 구체적으로
지정해서 식별을 하자.

Block : 독립적으로 존재할 수 있는 큰 덩어리
header, nav, card, product-card, login-form, profile, footer, ...

<article class="product-card">
	<div class="product-card__image">상품 이미지</div>
	<h2 class ="product-card__title">무선 마우스</h2>
	<p class="product-card__desc">조용한 사무용 마우스</p>
	<p class="product-card__price">29,000</p>
	<button class="product-card__btn">구매하기</button>
</article>

Element : Block 안에 포함되는 구성요소
- Element는 혼자 존재하기 보다는 bolck안에서 의미가 있다

Modifier : Block이나 Element의 상태 또는 변형을 의미한다. 
ex) product-card product-crd--sale
