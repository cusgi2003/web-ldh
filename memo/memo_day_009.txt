생성자

	객체를 메모리에 올리는 역할
	필드를 초기화하는 역할


메서드 

	만들고 싶은 기능
	자주 사용하는 기능을 만드는 것


this

	호출한 객체 자신
	this.필드명
	this()

	
super

	부모클래스
	super.필드명
	super()
	super.메서드명()
	

오버라이딩

	메서드 재정의
	상속관계에서만 사용할 수 있음
	
	부모클래스에 있는 메서드 이름으로 사용하고 싶지만
	자식클래스의 상황에 맞지 않아서 자식클래스에서 재정의할 때

	접근제한자, 반환형, 메서드명, 매개변수는 그대로 사용
	내용만 수정

	@Override -> 어노테이션
		컴파일러한테 오버라이드된 메서드라고 알려주는 것


클래스 타입변환
	
	자동타입변환

		자식객체 -> 부모타입

	강제 타입 변환

		부모타입 -> 자식타입
		(자식클래스) 부모객체


다형성
	
	상속 + 오버라이딩 + 타입변환

	부모타입으로 자식객체를 다룰 수 있는 것
	자식객체마다 다른 성능을 낼 수 있다
	코드를 수정하지 않고 기능을 확장할 수 있다
