반응형 웹
- 하나의 html 구조로 다양한 화면 크기를 지원한다.

픽셀 대신 비율, vw, vh, em, rem같은 단위를 사용한다

max-width : 100%, height : auto; 로 이미지가  부모 요소 너비에 맞게 줄어들게 한다
미디어쿼리를 활용하여 특정 화면 크기 이상 또는 이하에서만 스타일을 적용한다.


모바일 퍼스트 원칙
- 모바일에서 먼저 디자인하고, 큰 화면으로 확장한다
- css를 작성할 때 모바일 기준으로 기본 스타일을 먼저 작성하고, 화면이 커질때만 스타일을 추가한다

@media(조건){
	스타일
}

너비가 최소 768px 이상이 되면 스타일을 적용
@meida(min-width : 768px){
}
