grid 레이아웃
- 화면을 가로줄과 세로줄이 있는 표처럼 나누고, 그 안에 요소를 배치하는 레이아웃 방식이다.

flex vs grid
flex ->한 줄 방향 배치에 강점이 있다
grid -> 행과 열을 동ㄱ시에 다루는 2차원 배치에 강하다

grid container : grid가 적용된 부모 요소
gird item : 컨테이너 안에 있는 자식 요소들
cell : 행과 열이 만나는 한 칸
grind line : 칸을 나누는 선
grid tack : 행 또는 열 하나를 의미
gap : 칸 사이의 간격
grid area : 여러 칸을 합친 영역
