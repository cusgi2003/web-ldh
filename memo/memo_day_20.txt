절대경로

	윈도우를 기반으로 경로를 설정

상대경로

	현재 프로젝트 기반으로 경로를 설정


입출력 : 데이터를 읽거나, 내보내는 행위

	input : 데이터를 외부에서 읽어오는 것
	output : 데이터를 외부로 내보내는 것

바이트 기반의 스트림

	데이터를 바이트 단위로  읽거나 내보내는 스트림

	FileInputeStream

		외부에서 파일의 내용을 읽어오는 클래스
		생성자에 경로(FileNotFoundException 예외가 발생할 수 있다)

	read() : 데이터를 1바이트 읽어와 int로 저장	

	한글 같은 경우는 한글자에 1바이트가 아니기 때문에 바이트기반 스트림으로 
	읽어오기가 쉽지 않다

	read(byte[] b)를 통해서 읽어온  내용을 바이트 배열로 저장한다
	new String(byt[])문자열 생성자를 통해 문자열을 반환한다

	장점
		ㄴ 모든 파일을 처리할 수 있다.(이미지, 영상, 텍스트 등)
		ㄴ 인코딩 영향을 안받는다
		ㄴ 바이너리 데이터 처리에 필수
			ㄴ 이미지, 영상, 실행파일 << 같은 것들은 문자로 처리 불가
