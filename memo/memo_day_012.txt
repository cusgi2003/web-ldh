예외처리

	예외들이 클래스 형태로 존재
	Exception클래스의 자식클래스

	try{
		예외가 발생할 수 있는 코드
	}catch(매개변수){
		예외가 발생하면 실행할 코드
	}finaly{
		예외가 발생하든 안하든 무조건 실행하는 코드
	}


메서드 던지기
	
	메서드 내부애서 예외가 발생했을 때, 내부에서 처리하는게 아니라
	메서드를 호출한 쪽에서 처리한다


Object클래스(모든 클래스의 최상위 클래스)

	equals() : 주소가 같은지 비교
	hashcode() : 주소를 반환
	toString() : 화면에 출력을 어떻게 할건지

	대부분의 클래스들이 오버라이딩하여 위 메서드들을 사용
