반복문
- 같은 작업을 여러번 반복 실행할 때 
- 프로시저 안에서만 작성 가능

WHILE 
begin
  변수 선언
  while 조건 do
      반복실행할 문장
      set 으로 변수값 변경
  end while
end

REPEAT : do -while과 사용법이 같다.
repeat
    반복실행할 명령;

until 조건; -> 조건이 참이 되면 종료
end repeat

LOOP : 무한루프 기반, 종료조건을 명시해야 한다.
- 라벨을 달아야 한다.

라벨:loop
    반복 실행할 문장;
    if 조건 then
        leave 라벨;
    end if;
end loop;

ITERATE -> contiue
- 현재 반복을 중단하고 다음 반복으로 바로 이동하는 명령어

라벨 : loop
	if 조건 then
		iterate 라벨;
	end if;
end loop;

MySQL의 예외처리
Handler
- Procedure 안에서 에러, 경고, 조회 결과 없음 같은 상황을 처리하는 예외처리 문법
- 프로시저 실행중 문제가 생겼을 때 프로시저가 바로 중단되지 않도록 처리 방법을 정해두는 문법


기본 문법
DECLARE 핸들러타입 HANDLER FOR 예외
BEGIN
예외처리
END;

핸들러 타입 종류
CONTINUE : 에러 처리 후 다음 문장 계속 실행
EXIT : 에러 처리 후 BEGIN - END 블록 종료

예외종류
SQLEXCEPTION : 대부분의 SQL 에러
SQLWARNING : SQL 경고
NOT FOUND : 조회 결과 없음
MySQL 에러코드 : 1062 (중복 키 에러)
