집계 함수(그룹 함수)
ㄴ 하나 이상의 행에 대한 값들에 대해 집계하여 1개의 값으로 반환
ㄴ 일반적으로는 일반 속성과 집계함수를 SELECT 절에서 같이 사용할 수는 없다
ㄴ NULL값은 집계하지 않는다


group by
	같은 데이터를 묶어 집계할 때 사용하는 문법
	부서명, 상품명, 카테고리별

having
	group by로 만든 그룹을 조건으로 필터링할 때 사용하는 문법
	where은 전체 행에 대해서 필터링
	having은 그룹핑을 한 다음 필터링

select -> from -> where -> group by -> having

ROLLUP
	결과에 중간합계와 총 합계까지 자동으로 추가해주는 기능


order by 
	조회 결과를 원하는 순서로 정렬할 때 사용
	ASC : 작은값 -> 큰값
	DESC : 큰값 -> 작은값

오름차순 
	숫자 : 작은값 -> 큰 값
	문자 : ㄱ -> ㅎ, A -> Z
	날짜 : 오래된 날짜 -> 최신 날짜

order by 정렬컬럼1 정렬방식, 정렬컬럼2 정렬방식 ... 


순위를 정하는 함수
	ROW_NUMBER() : 무조건 1,2,3,4 처럼 번호를 붙인다
	RANK() : 공동 순위를인정하고, 다음 순위를 건너뛴다
	DENSE_RANK() : 공동 순위를 인정하지만, 다음 순위를 건너뛰지 않늗나
	NTILE(n) : 데이터를 n개의 그룹으로 나눈다
	PERCENT_RANK() : 상대적인 순위 비율을 구한다

순위함수() OVER(ORDER BY 정렬기준 컬럼 정렬기준)
