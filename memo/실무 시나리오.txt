A 온라인 쇼핑몰은 회원 정보를 관리하는 시스템을 운영하고 있다.

현재 개발팀은 회원 테이블의 gender 컬럼에 인덱스를 생성할지 검토 중이다.

회원 테이블의 전체 데이터 수는 다음과 같다.

전체 회원 수 : 10,000명

gender 컬럼의 데이터는 다음과 같이 저장되어 있다.

gender 값   인원 수

M   5,200
F   4,800

개발팀은 다음 기준으로 인덱스 적용 여부를 판단하려고 한다.

인덱스 컬럼의 분포도가 10~15% 이내인 경우 인덱스 효율이 높다고 판단한다.
문제

위 gender 컬럼의 분포도를 계산하고, 해당 컬럼에 인덱스를 적용하는 것이 적절한지 판단하시오.

조건
컬럼값의 종류 수를 먼저 계산한다.
분포도 계산 과정을 작성한다.
인덱스 적용 적절 여부를 작성한다.
이유를 함께 작성한다.
단순히 “비효율적이다”라고만 작성하면 오답 처리한다.

작성해야 하는 내용
컬럼값의 종류 수 : 2가지(M/F)
분포도 계산 : (1/2) x 100 = 50%
인덱스 적용 여부 : 부적절
판단 이유 : 인덱스 분포도를 크게 초과
