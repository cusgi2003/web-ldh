 
## 운영체제란 무엇인가
- 컴퓨터의 하드웨어와 소프트웨어 기능을 사용자가 편리하게 사용할 수 있도록 돕는 "시스템 소프트웨어"
- 컴퓨터 하드웨어와 사용자 사이에서 자원을 효율적으로 관리하고 배분하는 '총괄 관리자' 역할을 수행하며, 
시스템의 성능(처리 능력, 응답 시간, 신뢰도, 가용성)을 결정짓는 핵심 요소
- 실무 개발에서는 내가 만든 프로그램이 어떤 운영체제 위에서 작동할지 파악하는 것이 필수적이다
- 정보 자산 목록과 서버 구성도를 대조하여 운영 체제의 종류와 버전을 식별해야 한다
```
    정보 자산(Information Asset): 기업이나 기관이 보유한 서버, 소프트웨어, 데이터 등 가치 있는 IT 자원 전체

    커널(Kernel): 하드웨어(CPU, 메모리, 주변기기 등)와 응용 프로그램(소프트웨어) 사이에서 자원을 관리하고 명령을
    전달하는 가장 근본적인 프로그램 핵심(Core)을 뜻하는 영어 단어에서 유래했으며,
    항상 메모리에 상주하며 시스템을 제어
  ```
- 운영체제를 이해하지 못하면 프로그램이 시스템 자원을 과하게소모하거나, 특정 환경에서 실행되지 않는 문제가 발생할 수 있다.
