``` sql
  ABS() -> 절대값 함수

  ROUND(숫자, 자릿수) -> 반올림

  FLOOR() -> 주어진 숫자보다 작거나 같은 정수 중 최대값 반환

  TRUNCATE() -> 소수점 버림

  CEIL() -> 주어진 숫자보다 크거나 같은 정수중 최소값 반환

  SIGN()
  인자로 전달된 값이 양수면 1, 음수면 -1, 0이면 0, NULL이면 NULL

  MOD() -> 나누기한 나머지 값

  POWER(밑값, 지수) -> 밑값 * 지수

  RAND() -> 0 이상 1 미만 난수 반환

  GREATEST(), LEAST() -> 최대값, 최소값 반환

  FORMAT(숫자, 소수점 자리수) -> 숫자를 사람이 보기 좋게 '문자열 형태로 포맷팅'하는 함수

  COUNT() -> 값의 개수를 반환하는 함수

  SUM()

  AVG()

  MAX()

  MIN()

```
