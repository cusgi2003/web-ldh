## OSI 7계층 (Open System Interconnection)
### 개념 
| 컴퓨터끼리 통신할 때 어떤 일을 7단계로 나눠서 처리하는 구조  
▷ 통신 구조를 쉽게 이해하기 위해  
▷ 문제 발생 시 원인 파악  
▷ 장비와 기술 역할 구분  

| 계층 | 이름 | 핵심 역할 |
| --- | --- | --- |
| 7 | 응용 | 사용자 직접 사용하는 서비스 |
| 6 | 표현 | 데이터를 서로 이해할 수 있게 변환 |
| 5 | 세션 | 컴퓨터끼리 연결을 만들고 유지하거나 끊는 역할 |
| 4 | 전송 | 내 데이터를 안전하게 보내는 핵심 계층 |
| 3 | 네트워크 | 목적지까지 가장 안전하고 빠르게 IP주소를 보내는 계층 |
| 2 | 데이터링크 | 같은 네트워크 내부로 전달 / MAC 기반 전달 |
| 1 | 물리 | 데이터를 0과 1의 전기신호로 변환하여 보내는 계층 |

```
  데이터 전송 흐름(요청 시)  
  7 -> 6 -> 5 -> 4 -> 3 -> 2 -> 1
```

```
  데이터 수신 흐름
  1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7
```
