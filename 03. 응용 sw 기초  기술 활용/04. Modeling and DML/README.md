## 데이터 모델링
### 개념
- 현실 세계의 데이터를 데이터베이스로 옮기기 위해 구조화하는 작업
```
  추상화 : 현실세계를 데이터로 표현할 때 중요한 요소만 선택해서 표현한다
  단순화 : 누구나 이해할 수 있도록 쉽게 표현해야 한다
  명확화 : 의미가 모호하지 않도록, 오해없이 해석되게 만든다
```

## DML
### 1. SELECT (조회)
``` sql
  SELECT 컬럼명1, 컬럼명2, ...
  FROM 테이블명
  WHERE 조건식;
```
#### 1-1 NULL 비교
``` sql
  -- 컬럼 값이 NULL인 행만
  WHERE 컬럼명 IS NULL;
  -- 컬럼 값이 NULL이 아닌 행만
